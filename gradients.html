Introduction to Gradients

What is a gradient?

You may be familar with functions in 2D space. For example, you might have seen this function before: \(f(x) = x^2\). We can represent inputs and outputs in a 2D (flat) space.

A fundamental aspect of gradients is that they typically deal with dimensions higher than 2D.
For example, this function \(f(x, y) = x^2 + y^2\) is a function in 3D space. We have two inputs that can be represented on our previous 2D plane, and one output that can be represented on a third axis. So, this new third up-down axis is simply the output of the function just as the y-axis was the output of the 2D function.
